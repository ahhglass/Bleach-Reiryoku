### Über den Mod
**Bleach: Reiryoku** ist ein Mod, der das Bleach-Universum nach Hytale bringt. Erstelle legendäre Waffen, nutze spirituelle Kräfte und entdecke neue Items und Dekorationen.

> ⚠️ **Früher Zugang:** Dieser Mod befindet sich in aktiver Entwicklung. Funktionen können sich in zukünftigen Updates ändern.

### Funktionen
- **Seelenerinnernde Klingen** — erwecke die Kraft der Zanpakutō
- **Der Shinigami-Weg** — vom Asauchi zur wahren Form
- **Arsenal der Soul Society** — craften, handeln, kämpfen
- **Geist der Gotei 13** — dekoriere deine Welt mit Truppenbannern

### Installation
1. Lade den Mod von [CurseForge](https://www.curseforge.com/hytale/mods/bleach-reiryoku) herunter
2. Platziere die Mod-Datei im `mods`-Ordner von Hytale
3. Aktiviere den Mod in den Welteinstellungen
4. Starte das Spiel und finde neue Werkbänke
